# 📜 Commands

## 👤 Player Commands

| Command                  | Description      |
| ------------------------ | ---------------- |
| `/lifesteal`             | Open main GUI    |
| `/lifesteal stats`       | View your stats  |
| `/lifesteal leaderboard` | View top players |
| `/lifesteal revive`      | Open revive menu |

Alias:

```
/ls
```

---

## 👑 Admin Commands

| Command                             | Description                 |
| ----------------------------------- | --------------------------- |
| `/lsadmin set <player> <hearts>`    | Set player hearts           |
| `/lsadmin add <player> <amount>`    | Add hearts                  |
| `/lsadmin remove <player> <amount>` | Remove hearts               |
| `/lsadmin revive <player>`          | Force revive player         |
| `/lsadmin reload`                   | Reload plugin configuration |

Permission:

```
lifesteal.admin
```

Default: **OP**
